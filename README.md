# Description

This is a default development environment made with Gulp 3 that minify CSS JS and Images.

## Requirements

*NodeJs check [Node](https://nodejs.org/en/) for download the latest version.

*Gulp 3

```
npm install gulp@3
```

## Installation

Run

```
npm install
```
